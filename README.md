# 📒 Notes App (Android + Firebase)

## 📌 Description
This is a simple Notes App built using Android Studio and Firebase.  
It allows users to register, login, and manage notes in real-time.

---

## 🚀 Features
- User Authentication (Login & Signup)
- Add Notes
- Edit Notes
- Delete Notes
- Real-time sync using Firebase
- Logout functionality

---

## 🛠️ Technologies Used
- Java (Android)
- Firebase Authentication
- Firebase Realtime Database
- XML (UI Design)
